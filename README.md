# CH55x-Keil-Plugin-
用于keil的南京泌恒CH55x资料芯片下载支持插件

功能说明：本组件为Keil外挂下载组件，用于在Keil中直接下载CH55x系列芯片（目前支持CH551/CH552/CH553/CH554）
目前仅支持USB接口的官方ISP下载，不支持串口下载。
Keil4/Keil5可以通用
不支持在线调试（因为芯片本身不支持）

编译环境win10 x64 vs2015
具体兼容性情况等待反馈。


本项目开源，但是源码不直接公开。
